- Vue3、TypeScript、Vite、canvas-confetti

### 使用说明
```
# 克隆项目
git clone https://gitee.com/xiaoxiang_reincarnation/vue3-ts-glgg.git

# 进入项目
cd vue3-ts-glgg

# 安装依赖
npm install

# 运行项目
npm run dev

# 打包发布
npm run build
```
